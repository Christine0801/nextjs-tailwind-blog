# 三大会议

## 自律

**任何报会短信的冒号均为中文格式，不是英文 ，涉及跟老总有关系的短信，出现错误，都是模式 100，5 人以上在场，必须提前一小时报自律短信**

1. 晚上会场报会短信，每天晚上 7：00-7：20 发**（如果特殊情况不开会场不报短信，应提前跟管自律短信汇报情况）**

```txt
哥哥\姐姐你好，我叫XXX，是XXX的业务员，今天晚上8：00-9：00订工作，9：00-10：00组织学习，在XXXXXX，共计X人，麻烦哥哥\姐姐记一下，谢谢，哥哥\姐姐辛苦了。
```

2. 早会短信

```txt
哥哥\姐姐你好，我叫XXX，是XXX的业务员，今天早上7：00-8：00订工作，8：00-9：00组织学习，在XXXXXX，共计X人，麻烦哥哥\姐姐记一下，谢谢，哥哥\姐姐辛苦了。
```

3. 到人短信

**分两种情况：**如果是**8：30 之前到**，就需要**提前一天**报短信，8：30 到就当天发，短信在每天晚上**8：30-9：00**发

- 8：30 之前到：

```txt
哥哥\姐姐你好，我叫XXX，是XXX的业务员，我的业务\旁系XXX明天到新人，安排住在XXXXXX，麻烦哥哥\姐姐记一下，谢谢，哥哥\姐姐辛苦了。
```

- 8：30 之后到：

```txt
哥哥\姐姐你好，我叫XXX，是XXX的业务员，我的业务\旁系XXX到新人，安排住在XXXXXX，麻烦哥哥\姐姐一下，谢谢，哥哥\姐姐辛苦了。
```

4. 融入**（在融入召开之前，提前一小时报）**

```txt
哥哥\姐姐你好，我叫XXX，是XXX的业务员，今天中午\晚上11：00-12：00在XXXXXX搞融入，共计X人，麻烦哥哥\姐姐记一下，谢谢，哥哥\姐姐辛苦了。
```

## 新晨新管

### 相关短信

新升经理，在当月**一号**必须报申请承办短信：

```txt
哥哥\姐姐你好，我叫XXX，是XXX的业务员，我申请承办新晨新管，麻烦哥哥\姐姐安排一下，谢谢，哥哥\姐姐辛苦了。
```

自己家有新人加入，给他报申请过新晨新管短信：

```txt
哥哥\姐姐你好，我叫XXX，是XXX的业务员，我的业务员\旁系XXX申请过新晨新管，麻烦哥哥\姐姐安排一下，谢谢，哥哥\姐姐辛苦了。
```

模拟新晨新管**（次日中午 12：00-12：20）**短信：

```txt
哥哥\姐姐你好，我叫XXX，是XXX的业务员，今天中午1：00-1：20在XXXXXX模拟新晨新管，共计X人，麻烦哥哥\姐姐记一下，谢谢，哥哥\姐姐辛苦了。
```

参会短信**（次日 9：00-12：00）**，**··**表示两个**空格**

```txt
哥哥\姐姐你好，我叫XXX，是XXX的业务员，谢谢哥哥\姐姐安排··XXX··XXX··XXX··过新晨新管。
地址：XXXXXX
立岗：XXX
读管：XXX
跟会经理：XXX··XXX
值勤经理：XXX··XXX
参加人员：XXX··XXX··XXX··XXX
共计X人，麻烦哥哥\姐姐记一下，谢谢，哥哥\姐姐辛苦了。
```

### 流程

1. **承办人：**
   8：30 之前收到通知，须当晚找好人，并报参会短信给老总
   8：30 之后收到通知则不用
   分配新人对应直线经理或伞尖找好要的人，当晚写好会单（2 张）

2. 次日早上 9：00-12：00 给老总报参会短信，准备好地图、笔、《生活管理二十条》、胶布

   12：00 之前通知模拟时间、地点
   三次通知正式过：8：30-9：00 通知正式过的时间和地点、12：00-12：30 再次确认正式过的时间和地点、4：30-5：00 通知立岗、跟会、值勤、读管提前十分钟到
   12：00-12：20 报自律短信，模拟结束给负责新晨新管老总打电话反映模拟情况（3：00）

3. 1：00 之前到达会场布置，写好台词、引字形介绍、注意事项 、目标、自报家门、高素质商人
   解释会场、台词、注意事项
   开始模拟（3 遍及 3 遍以上）
   **300 份经理到现场监督模拟**

4. 注意事项：
   手机静音、直上直下、不准抢上、不准踢跨凳子、认真严肃、禁止现场复制、台上站 1 位，台下站 2 位
   中途可休息 2~5 分钟让新人记台词、反应情况
   模拟时间 1：00-3：00||5：00-6：00
   提前一天晚上给新人复制台词
   结束后把单子交给承办人

## 素质

### 相关短信

新升经理，在当月**一号**必须报申请承办短信：

```txt
哥哥\姐姐你好，我叫XXX，是XXX的业务员，我申请承办素质，麻烦哥哥\姐姐安排一下，谢谢，哥哥\姐姐辛苦了。
```

自己家的人过素质只需本子写好检查完没问题交上去，本子没问题过了之后，就等排队，通知过

模拟素质（八大心态）**（次日中午 12：00-12：20）**短信：

```txt
哥哥\姐姐你好，我叫XXX，是XXX的业务员，今天中午1：00-1：20在XXXXXX模拟素质（八大心态），共计X人，麻烦哥哥\姐姐记一下，谢谢，哥哥\姐姐辛苦了。
```

参会短信**（次日 9：00-12：00）**，**··**表示两个**空格**

```txt
哥哥\姐姐你好，我叫XXX，是XXX的业务员，谢谢哥哥\姐姐安排··XXX··XXX··XXX··过素质（八大心态）。
地址：XXXXXX
立岗：XXX
跟会经理：XXX··XXX
值勤经理：XXX··XXX
参加人员：XXX··XXX··XXX··XXX
共计X人，麻烦哥哥\姐姐记一下，谢谢，哥哥\姐姐辛苦了。
```

### 流程

1. 收到老朋友通知、短信，立即通知分配找人，当晚写会单（2 张）
2. 次日早上 9：00-12：00 给管素质的老朋友报参会短信，并通知相应伞尖、直线经理正式过和模拟的时间、地点
3. 1：00 前到达会场布置好会场，地图上写好自报家门、台词、引字形、注意事项、立岗名字、过的人名字
   注：模拟时亲自示范读本细节，并解释加快语速

## 能力

### 相关短信

新升经理，在当月**一号**必须报申请承办短信：

```txt
哥哥\姐姐你好，我叫XXX，是XXX的业务员，我申请承办能力，麻烦哥哥\姐姐安排一下，谢谢，哥哥\姐姐辛苦了。
```

自家人过能力，须至少模拟 3 遍及 3 遍以上，模拟好没问题可以过，在晚上**8：30-9：00**，把短信报给管能力的老朋友

```txt
哥哥\姐姐你好，我叫XXX，是XXX的业务员，我的业务\旁系XXX针对性工作XXX已复制好，复制人：XXX，已模拟3遍，麻烦哥哥\姐姐安排一下，谢谢，哥哥\姐姐辛苦了。
```

参会短信**（次日 9：00-12：00）**，**··**表示两个**空格**

```txt
哥哥\姐姐你好，我叫XXX，是XXX的业务员，谢谢哥哥\姐姐安排我的业务员\旁系XXX过针对性工作XXX，已复制好，复制人：XXX，已模拟3遍。
地址：XXXXXX
点评经理：XXX··XXX··XXX
值勤经理：XXX··XXX
共计X人，麻烦哥哥\姐姐记一下，谢谢，哥哥\姐姐辛苦了。
```

### 流程

1. 当晚 8：30-9：00 报过能力短信，等通知明天是否过
2. 收到通知明天过，当晚组织 3 人以上写会单
3. 根据通知时间（中午 1：00-1：20 或晚上 6：00-6：20 || 6：20-6：40）到会场布置、发短信
4. 通知过的人 直线经理买红牛提神，结束后上去了解情况
