# 三大会议

## 自律

**任何报会短信的冒号均为中文格式，不是英文 ，涉及跟老总有关系的短信，出现错误，都是模式 100，5 人以上在场，必须提前一小时报自律短信**

1. 晚上会场报会短信，每天晚上 7：00-7：20 发**（如果特殊情况不开会场不报短信，应提前跟管自律短信汇报情况）**

```txt
哥哥你好，我叫XXX，是XXX的业务员，今天晚上8：00-9：00订工作，9：00-10：00组织学习，在XXXXXX，共计X人，麻烦哥哥记一下，谢谢，哥哥辛苦了。
```

2. 早会短信

```txt
哥哥你好，我叫XXX，是XXX的业务员，今天早上7：00-8：00订工作，8：00-9：00组织学习，在XXXXXX，共计X人，麻烦哥哥记一下，谢谢，哥哥辛苦了。
```

3. 到人短信

​ **分两种情况：**如果是**8：30 之前到**，就需要**提前一天**报短信，8：30 到就当天发，短信在每天晚上**8：30-9：00**发

- 8：30 之前到：

```txt
哥哥你好，我叫XXX，是XXX的业务员，我的业务/旁系XXX明天到新人，安排住在XXXXXX，麻烦哥哥记一下，谢谢，哥哥辛苦了。
```

- 8：30 之后到：

```txt
哥哥你好，我叫XXX，是XXX的业务员，我的业务/旁系XXX到新人，安排住在XXXXXX，麻烦哥哥记一下，谢谢，哥哥辛苦了。
```

4. 融入**（在融入召开之前，提前一小时报）**

```txt
哥哥你好，我叫XXX，是XXX的业务员，今天中午/晚上11：00-12：00在XXXXXX搞融入，共计X人，麻烦哥哥记一下，谢谢，哥哥辛苦了。
```

## 新晨新管

### 相关短信

​ 新升经理，在当月**一号**必须报申请承办短信：

```txt
哥哥你好，我叫XXX，是XXX的业务员，我申请承办新晨新管/素质/能力，麻烦哥哥安排一下，谢谢，哥哥辛苦了。
```

​

​ 自己家有新人加入，给他报申请过新晨新管短信：

```txt
哥哥你好，我叫XXX，是XXX的业务员，我的业务员/旁系XXX申请过新晨新管，麻烦哥哥安排一下，谢谢，哥哥辛苦了。
```

​

​ 模拟新晨新管**（次日中午 12：00-12：20）**短信：

```txt
哥哥你好，我叫XXX，是XXX的业务员，今天中午1：00-1：20在XXXXXX模拟新晨新管，共计X人，麻烦哥哥记一下，谢谢，哥哥辛苦了。
```

### 流程

1. **承办人：**
   8：30 之前收到通知，须当晚找好人，并报参会短信给老总
   8：30 之后收到通知则不用
   分配新人对应直线经理或伞尖找好要的人，当晚写好会单（2 张）

2. 次日早上 9：00-12：00 给老总报参会短信，准备好地图、笔、《生活管理二十条》、胶布

   12：00 之前通知模拟时间、地点
   三次通知正式过：8：30-9：00 通知正式过的时间和地点、12：00-12：30 再次确认正式过的时间和地点、4：30-5：00 通知立岗、跟会、值勤、读管提前十分钟到
   12：00-12：20 报自律短信，模拟结束给负责新晨新管老总打电话反映模拟情况（3：00）

3. 1：00 之前到达会场布置，写好台词、引字形介绍、注意事项 、目标、自报家门、高素质商人
   解释会场、台词、注意事项
   开始模拟（3 遍及 3 遍以上）
   **300 份经理到现场监督模拟**

4. 注意事项：
   手机静音、直上直下、不准抢上、不准踢跨凳子、认真严肃、禁止现场复制、台上站 1 位，台下站 2 位
   中途可休息 2~5 分钟让新人记台词、反应情况
   模拟时间 1：00-3：00||5：00-6：00
   提前一天晚上给新人复制台词
   结束后把单子交给承办人
